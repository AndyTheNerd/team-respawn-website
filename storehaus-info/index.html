Storehaus RTS

A Discord bot built with discord.js v14 that provides information and statistics for Halo Wars 2, Age of Empires II and Age of Empires IV.

Features

Age of Empires IV

Get civilization statistics, search for players, and view game history. Data is fetched from aoe4world.com.

  • Civilization stats for 1v1, 2v2, 3v3, and 4v4 (Quick Match and Ranked)
  • Player search by name (minimum 3 characters)
  • Recent games for a player by profile ID

Age of Empires II

Look up units, technologies, structures, and civilizations from the game database. All searches are case-insensitive and support partial matching.

  • Units — cost, stats, armor, and more
  • Technologies
  • Structures
  • Civilizations — bonuses, unique units, and more

Halo Wars 2

The bot also includes an AI-powered strategy advisor for Halo Wars 2, using retrieval-augmented generation (RAG) and multiple AI providers (Groq, Cloudflare, OpenRouter, Cerebras) to answer strategy questions.

Commands

Slash commands are available for each game:

  • /aoe4 — Civilization stats by team size (1v1–4v4), player search, and recent games
  • /aoe2 — Unit, technology, structure, and civilization lookups
  • /ping — Replies with "Pong!"

The Halo Wars 2 strategy advisor is available via its own command in servers where the bot and the optional RAG service are configured.

Data Sources

  • Age of Empires IV: Statistics and player data from the aoe4world.com API
  • Age of Empires II: Game data stored locally in JSON format
